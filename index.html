식별자 뒤에 괄호()가 있으면 해당 식별자를 함수라고 부른다.
함수를 쓰는 이유는 프로그래밍 자체가 반복적으로 사용되는 코드를 자주 쓰는데
매번 동일한 코드를 반복해서 쓰는 것보다 함수로 한 번 정의하고 계속 사용하는 것이 효율적이기 때문이다.
함수 괄호 내부에 들어가는 것이 매개변수(parameter),
함수를 호출 할 때 넣는 값을 인수(argument),
함수의 결과값을 리턴값(return)이라고 한다.
예시===========================
def add(a, b): # a, b는 매개변수
return a + b
add(1, 3) # 1, 3는 인수
===============================
매개변수는 함수에 이미 입력되어 있고, 앞으로 전달될 값을 받는 변수를 의미하고
인수는 함수를 호출할 때 전달하는 입력값을 의미한다.
https://pjslog.tistory.com/3
함수에 넣는 인수의 위치가 정해져 있는 방식을 위치 인수(positional argument)라고 한다.
가변 인수(variable argument) 는 임의 개수의 인수를 받을 수 있다.
파이썬에서는 인수의 순서와 용도를 매번 기억하지 않아도 되도록 키워드 인수(keyword argument)라는 기능을 제공.
sep(separation)
영단어 그대로, 분리하여 출력한다. 다만 분리할 (갈라놓을 문자를 지정할 수 있다.) 이것을 구분자라고 한다.
end
end옵션을 사용하면 그 뒤의 출력값과 이어서 출력한다. (즉, 줄바꿈을 하지 않게 된다.)
인수에는 리스트나 튜플을 사용할 수도 있다.
리스트나 튜플 앞에 *(애스터리스크)를 붙여서 함수에 넣어주면 된다. (리스트 앞에 *를 붙이면 언패킹(unpacking)이 되기 때문)
딕셔너리 언패킹은 딕셔너리 앞에 ** (애스터리스크 두 개)를 붙여서 함수에 넣어주면 된다.
함수의 종류를 살펴 보다 보면 입력값이나 결과값이 있거나 없는 함수가 있다.
입력이 필요 없을 경우는 입력값에 상관 없이 결과 값이 정해져 있다거나 한 경우일테고,
결과 값의 경우 대입만이 필요한 상황이 거나 결과 값이 필요 없는 상황일 것이다.
위와같은 두 경우뿐만이 아니라 수없이 많은 경우가 있겠지만 말이다.
함수 내부에서만 사용하는 변수를 지역 변수라고 하고, 그 외에도 전역적으로 사용하면서 함수 내부에서도 사용할수 있는 변수를 전역 변수 라고 한다.
#결국 함수는 사용함에 따라, 코드가 길어짐에 따라 계속 늘어날텐데 그 때마다 변수가 전역 변수 밖에 없다면은, 변수끼리 이름이 같으면 안되기 때문에 점점 명명하기가 복잡해진다.
전혀 상관이 없는 서로 다른 함수 내부에서, 변수명을 고려해야 하는 것은 불필요 하기때문에, 지역변수를 사용하는 것이 독립성을 향상시키는데 도움이 될것이라 생각한다. 함수(function): 불려진 시점에 특정한 작업을 수행하며, 입력값과 출력값(반환값)을 가질 수 있습니다.
인자(argument): 함수를 호출할 때 전달하는 입력값입니다.
매개변수(parameter): 함수가 실행될 때 입력값이 들어올 변수입니다.
반환값(return value): 함수가 종료될 때 호출 지점으로 전달할 출력값입니다.
변수(variable): 값을 가리키는 이름입니다.
스코프(scope): 변수가 유효한 범위입니다.
연산자(operator): 주어진 값들에 대해 정해진 연산을 수행합니다.
수리 연산자(mathematical operator): +, -, *, /, //, **
비교 연산자(comparison operator): ==, !=, <, >, <=, >=, is
논리 연산자(logical operator): and, or
소속 연산자(membership operator): in
제어문(control statements): 코드 블록의 흐름(실행 여부, 반복)을 제어합니다.
if: 명제가 참이면 실행합니다.
else: if 명제 이외의 경우에 실행합니다.
elif: if 명제 이외의 경우에 또 다른 명제가 참일 경우에 실행합니다.
while: 명제가 참일 동안 반복합니다.
for: 주어진 값들 하나씩 반복합니다.
자료형(data types): 값들의 종류를 나타냅니다.
정수(int), 부동소수점 수(float), 불리언(bool), 문자열(str), 튜플(tuple), 리스트(list), 딕셔너리(dict)
